# 1000번 문제
Scanner 사용 방법을 알지 못함.
1. 먼저 Scanner 객체명 = new Scanner(System.in) 선언해야하며 System.in 필수로 넣어줘야함
2. System.in 은 사용자로부터 입력을 받기 위한 입력 스트림이다.
그렇기 때문에 Scanner 뿐만 아니라 다른 입력 방식들도 사용자로부터 입력을 받기 위해서는 System.in 이 들어간다.
